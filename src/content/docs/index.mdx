## Core Principles

<CardGrid stagger>
	<Card title="Verified Senders" icon="checklist">
		VAMP binds message delivery to authenticated sender identity instead of anonymous origin claims.
	</Card>
	<Card title="Cryptographic Chain of Custody" icon="document">
		Identity First Mail preserves accountable message handling across the delivery path instead of trusting loose relay metadata.
	</Card>
	<Card title="Sender-Side Marginal Cost" icon="graph">
		The protocol is designed to make abusive sending economically and operationally harder at the sender edge.
	</Card>
</CardGrid>
